# No options to Sync a OneNote Notebook to an on-premises SharePoint Server

[Adam R -
MSFT](https://social.msdn.microsoft.com/profile/Adam%20R%20-%20MSFT)
3/20/2014 9:53:00 AM

-----

<div id="content">

**ISSUE:**

When Syncing OneNote 2013 notebooks to SharePoint Server for the first
time you do not have the option to save these to an on-premises
SharePoint Site.

  
**RESOLUTION:**

You will need to first save to "My Computer" and input the SharePoint
server URL. After this you will have a new location titled "Other Web
Locations".

  
When you open OneNote 2013 for the first time it will prompt you to
connect to the cloud. You can close this with the X in the corner if you
do not want to connect to the cloud.

[![
](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/0564.1.png)](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/0564.1.png)

  
Select your update Settings.

[![
](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/6038.2.png)](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/6038.2.png)

  
When trying to open a Notebook from SharePoint Server you do not have
any options for an on-premise site.  You only have SharePoint Online or
OneDrive. 

[![
](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/5621.3.png)](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/5621.3.png)

  
This also happens when you attempt to create a new Notebook. You need to
select "Computer", Type a name for your new Notebook and click on
"Create in a different folder".

[![
](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/2804.4.png)](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/2804.4.png)

  
Type in the URL of your SharePoint Site.

[![
](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/0880.5.png)](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/0880.5.png)

Browse to the Document Library where you would like to store your new
OneNote Notebook, give your Notebook a name and click "Create".

[![
](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/1817.6.png)](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/1817.6.png)

  
You now have a new option called "Other Web Locations" when attempting
to open or create new Notebooks.

[![
](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/0602.7.png)](media/TNBlogsFS/prod.evol.blogs.technet.com/CommunityServer.Blogs.Components.WeblogFiles/00/00/01/00/97/0602.7.png)
